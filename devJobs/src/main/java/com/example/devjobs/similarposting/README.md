### GPT API란?
```
GPT 프롬프트에 명령을 내리는 API
(수업자료에있는 PPT를 보세요)
```

### GPT API 설정
```
API KEY 발급
이것도 ppt를 보세요
```

### GPT API 사용방법
```
질문을 잘 작성해서 전송하고 답변을 받으면 됨

역할을 부여하는 이유:

GPT 역할:
요청할 내용에 따라 역할을 부여하면, gpt가 역할에 감정이입하여 답을 해준다

청자 역할:
답변을 보는 사람의 수준을 알려주면 그에 맞게 답변을 변경해준다.

더 자세한 답을 듣기 위해서 사용함 (생략 가능)

예시:
당신은 세계에서 가장 뛰어난 마케터입니다. 
저는 마케팅에대해 하나도 모르는 사람입니다. 
가능한 쉽게 마케팅 방법을 알려주세요.
```

### API 요금
```
GPT API를 유료서비스로, 사용하는 만큼 비용이 발생됩니다
비용은 토큰단위로 계산됩니다
GPT4버전보다 3.5버전이 더 저렴합니다
그리고 사용량에 비해 요금이 비싸지 않기 때문에 크게 걱정하지 않으셔도 됩니다

GPT3.5 모델 기준 비용:
https://platform.openai.com/docs/models/gpt-3-5-turbo
입력: $0.50 / 1M tokens
출력: $1.50 / 1M tokens
*1M는 백만개의 토큰

예를 들어 
입력: 자바가 뭐야?
출력: 자바(Java)는 오라클(Oracle)에서 개발한 객체지향 프로그래밍 언어로, 다양한 운영체제에서 사용할 수 있으며 특히 웹 애플리케이션 개발 및 모바일 앱 개발에 많이 사용됩니다. 또한 자바는 가상 머신(Java Virtual Machine, JVM)을 통해 다양한 플랫폼에서 실행될 수 있는 특징을 가지고 있습니다. 현재 자바는 업계 표준으로 자리 잡아있어서 많은 기업과 개발자들이 사용하고 있습니다. 
총 사용한 토큰량: 205 

API 사용량은 콴리 콘솔에서 확인하세요!
https://platform.openai.com/usage
대시보드 > usage
```
